# Launchyourproduct.de — Claude Kontext

## Projekt
Landing-Page für "Launch Your Product" — eine deutschsprachige Marketing/Produkt-Launch-Website.

## Repository
- **GitHub:** kmdigitallab/Launchyourproduct.de
- **Standard-Branch:** `main`
- **Sprache:** Deutsch (User kommuniziert auf Deutsch)

## Tech Stack
- Reines HTML/CSS (kein Framework)
- Font: Montserrat (Google Fonts)
- Primärfarbe: `#1a73e8` (Google-Blau)

## Dateistruktur
- `style.css` — globales Stylesheet (Hero, Cards, Form, Footer, Responsive)
- `CLAUDE.md` — diese Datei

## Design-Präferenzen
- Der User mag **Glassmorphism**-Stil (backdrop-filter, blur, transparente Karten)
- Animierte Gradient-Texte, Glow-Effekte, floating Blobs im Hintergrund
- Moderne, dunkle Hintergründe mit farbigen Akzenten (lila, pink, cyan)

## Wichtige Hinweise für neue Sitzungen
- User arbeitet im **Claude Code Web-Chat** (nicht CLI), daher funktioniert `/help` nicht
- Jede Sitzung startet frisch — diese Datei gibt den Kontext zurück
- Branch `claude/github-web-dev-skills-lzl7cx` existiert aus einer früheren Sitzung
- User nutzt möglicherweise ein **21dev**-Addon (noch nicht verifiziert/eingerichtet)
- Beim Start: kurz den aktuellen Branch-Stand prüfen und User begrüßen

## Workflow
1. Änderungen auf Feature-Branch entwickeln
2. Committen & Pushen
3. User entscheidet über Merge in `main`
